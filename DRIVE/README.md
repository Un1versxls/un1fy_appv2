# DRIVE

A productivity and habit-building iOS app built with SwiftUI.

## Features

- Onboarding flow with goal selection and plan choice
- Niche selection to personalize your tasks
- Daily tasks with points and streaks
- Progress tracking and gamification
- Profile management and settings

## Requirements

- iOS 16.0+
- Xcode 15+
- Swift 5.9

## Building

Open `DRIVE.xcodeproj` in Xcode and hit Run. You can also use the provided `scripts/build.sh` to create an archive.

## Sideloading

See `docs/SIDELOADING.md` for instructions on installing on a physical iOS device.

## Estimated Completion Time

All core features have been implemented. The remaining tasks (project configuration, build scripts, documentation) are estimated to take **3-4 hours** for a single developer to finalize, test, and polish. The current codebase provides a solid foundation for the app to run and be iterated upon.
